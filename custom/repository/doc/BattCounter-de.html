Battery Counter

Development release

Akkuzyklen-Zähler mit automatischer Erkennung anhand von Kapazitätssprüngen.

  • manuelle Auswahl möglich
  • modellübergreifende Statistiken
  • Akkus lassen sich je Modell nach Zellenanzahl und Kapazität filtern
  • Logdatei, in der jeder Zyklus eingetragen wird

Quelltexte und weitere Informationen sind in meinem GitHub Repository aufgeführt.

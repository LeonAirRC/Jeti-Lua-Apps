Beep Editor

Das Ändern von Tönen, Audiodateien und Vibrationen der Timersignalisierung ist normalerweise nur über eine USB-Verbindung möglich.
Diese App bietet eine einfache Oberfläche, um solche Elemente hinzuzufügen, zu bearbeiten und zu löschen.

Großartiges Video von Harry Curzon

Datei wählen

Die Jeti Sender haben bis zu vier Dateien, welche verschiedene Signalisierungsmuster beschreiben. Hier kann das zu bearbeitende Muster gewählt werden.

Editor

Jedes Element hat die Attribute Zeit [s] and Typ.

  • Typ 1 ist ein Ton und hat die Attribute Frequenz [Hz], Anzahl and Länge [ms]
  • Typ 2 beschreibt eine Audiodatei

Mit den Funktionstasten können Elemente hinzugefügt, getauscht und gelöscht werden.

Vibration

Nur sichtbar auf kompatiblen Sendern. Auf dieser Seite kann das Vibrationsprofil für jedes Element eingestellt werden.


Quelltexte und weitere Informationen sind in meinem GitHub Repository aufgeführt.

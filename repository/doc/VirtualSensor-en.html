Virtual Sensor

Calculates virtual telemetry values. The numeric inputs can be constants, telemetry values, proportional controls and switches. Numeric values are processed in a tree of mathematical operations that take either one or two parameters, eg ADD, MIN, SQRT, SIN, ABS.
The virtual sensor values can also be displayed in a telemetry frame.

Node types

  • constant: A constant integer. Non-integers can be created with a DIV of two integers.
  • sensor: telemetry values of the selected sensor
  • input: control sticks, switches, trainer inputs, PPM inputs, servo outputs
  • arithmetic operations: ADD, SUB, MUL, DIV, MIN, MAX, =, <, >, ABS, FLOOR, SQRT, Integral

Usage

  1. Create a virtual sensor.

  2. Set it's label, the decimal places and unit.

  3. Edit the tree structure.

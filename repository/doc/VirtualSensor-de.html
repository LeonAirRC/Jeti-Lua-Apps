Virtual Sensor

Berechnet einen virtuellen Telemetriewert und registriert virtuelle Logvariablen. Die numerischen Eingaben können Konstanten, Telemetriewerte, Proportionalgeber, Schalter, Schülereingaben, PPM Eingaben und Servo-Outputs sein. Numerische Eingaben werden in einem Baum aus mathematischen Operationen verarbeitet, die einen oder zwei Parameter besitzen, z.B. ADD, MIN, SQRT, SIN, ABS.
Ein virtueller Sensorwert kann außerdem in einem Telemetriefenster angegzeigt werden.

Knotentypen

  • konstant: Eine konstante Ganzzahl. Rationale Konstanten können mit einem DIV zweier Ganzzahlen erzeugt werden.
  • Sensor: Telemetriewert des gewählten Sensors
  • Eingabe: Proportionalgeber, Schalter, Schülereingaben, PPM Eingaben, Servoausgaben
  • Operatoren: ADD, SUB, MUL, DIV, MIN, MAX, ABS, ROUND, FLOOR, CEIL, SQRT, SIN, COS, TAN, ASIN, ACOS, ATAN

Bedienung

  1. Erstelle einen virtuellen Sensor.

  2. Stelle die Bezeichnung, die Anzahl von Dezimalstellen und die Einheit ein.

  3. Bearbeite die Baumstruktur.

  4. [Schließe die App und lade sie mit F(2) neu, um die neuen Logvariablen zu registrieren. Das Logging muss dazu vorher gestoppt werden.]

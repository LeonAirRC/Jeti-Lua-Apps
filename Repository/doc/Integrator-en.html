Vario Integrator

Announces the average climb rate in regular intervals.


Description

When piloting a glider it is essential to know not only the current climb rate, but also the average over a certain duration.
The app features two basic modes:

  • vario integrated: Actually integrates a variometer sensor value. The average then equals the integral divided by the interval duration.
  • altitude difference: Reads the altitude once before each announcement. The average is calculated as the difference of the current altitude relative to the last value, divided by the interval duration.

Usage

  1. Select a variometer or an altitude sensor as the input.

  2. Define a switch to turn the speech output on and off. When no switch is selected the output is always active.

  3. Set the interall (in seconds) at which the average climb rate is calculated and announced.
